# 《爆蛋行动》沙漠小镇·精美版 设计单

> 日期：2026-06-24　｜　把"积木版"沙漠小镇**升级成精美卡通版**：布局不变、用 CC0 模型、做活场景、**绝不穿模**。

---

## 1. 目标 / 不做

**做：** 保持现有 A/B 包点、中路、出生点的**布局与碰撞**；用 Kenney CC0 模型把场景做精美、做活；暖色沙漠午后光。
**不做：** 改布局；AI / 射击（另有 `2026-06-24-射击系统-design.md`）；联机。

## 2. 素材（全 CC0，记入 `docs/CREDITS.md`）

- **Kenney Survival Kit**：box / barrel / fence / tent / structure-canvas / signpost / rock-sand …（掩体 & 道具）
- **Kenney Nature Kit**：tree_palm* / cactus* / plant_bush* / cliff …（植被）
- 格式 **GLB**，用 three 自带的 **GLTFLoader**（无需新依赖）。
- 只把**用到的子集** GLB 拷进 `public/models/kenney/`，仓库不臃肿。

## 3. 硬规则：绝不穿模（用户特别要求）⚠️

1. **贴地**：放置前用包围盒求底部，平移使**底面正好 y=0**——不陷进地、不浮空。
2. **不重叠**：每个模型按包围盒占位，放置时与建筑/彼此**留空隙**（放置校验，发现重叠就挪开或不放）。
3. **看得见即挡得住**：每个**实心**模型（箱/桶/树/仙人掌/大石/帐篷）配一个与其 footprint 一致的 **AABB 碰撞盒**，玩家撞上被挡，**绝不穿过看似实心物**；仅极小的草/花无碰撞且**摆在路边不挡道**。
4. **不闪烁**：A/B 贴地标记垫高一丁点 + `polygonOffset`/`renderOrder`，避免与地面 z-fighting。
5. **验收**：浏览器**多角度截图巡检**，发现任何穿模/穿地/穿墙/闪烁就修，干净才算完。

## 4. 文件结构（新增 / 改）

- **新增** `src/game/world/modelLoader.ts`：加载并缓存 GLB；`clone` 出实例；`placeOnGround()` 贴地；可选按包围盒生成碰撞盒。**防穿模的核心工具。**
- **改** `src/game/world/desertMap.ts`：保留碰撞墙；木箱方块 → Kenney 木箱/油桶模型；加棕榈/仙人掌/帐篷/栅栏/石头/灌木点缀；大建筑用**干净土墙**（沙土色 + 门窗 + 屋顶）。
- **新增** `public/models/kenney/**`：用到的 GLB 子集。
- **改** `src/main.ts`：进场前 `preload` 模型（很小、很快；必要时加"加载中…"）。
- **改** `docs/CREDITS.md`：记两个 Kit（CC0）。

## 5. 帧率（守 ≥30、争 60）

低多边形；同款模型**加载一次、clone 复用**；阴影适度；`pixelRatio ≤ 2`。

## 6. 完成标准（DoD）

看着是**精美卡通沙漠小镇**（有植被、道具、暖光）；走位 / 撞墙 / 躲掩体与现在**完全一样**；**无任何穿模/穿地/穿墙/闪烁**；帧率 ≥ 30；CREDITS 记好授权。
